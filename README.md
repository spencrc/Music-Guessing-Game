# Music-Guessing-Game

This repository hosts the code used for [Drain Gangle](https://draingangle.com/).

## Project Setup

```sh
pnpm install
```

### Compile and Hot-Reload for Development

```sh
netlify dev
```

### Type-Check, Compile and Minify for Production

```sh
pnpm build
```

### Lint with [ESLint](https://eslint.org/)

```sh
pnpm lint
```
